# Get started

Zahir Design System is a reusable component library that helps Zahir Developers build UIs faster. The goal is to build durable UIs more productive and satisfy.

## Install

Zahir UI components are written in React, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df). It requires Storybook version 5.2-beta and up.

Add Zahir UI to your project.

`npm install --save zahir-ui`

#### Requirement

`npm install --save @fortawesome/fontawesome-svg-core@1.2.27 @fortawesome/react-fontawesome@0.1.8 @material-ui/core@4.9.1 @material-ui/icons@4.9.1 react-number-format@4.4.1 react-select@2.4.2`

### **Font**

Add this link in file index.html

```
import { global } from ‘zahir-ui’

<link href={global.fontUrl} rel="stylesheet" />
```
or
```
<link href="https://fonts.googleapis.com/css?family=Muli:400,500,600,700,800,900&display=swap" rel="stylesheet" />
```

### **Theming**

Customize Zahir UI with your theme. You can change the colors, the typography and much more.

```
import { ThemeProvider, theme } from ‘zahir-ui’

const myTheme = {
  // Primary color your app
  primary: '#006391',
  // Secondary color your app
  secondary: '#20BF6B',
  // Customize your theme base on Material-UI
  custom: {
    MuiButton: {
      root: {
        border: 4
      }
    }
  }
}

function App() {
  return (
    <ThemeProvider
      theme={theme(myTheme)}
    >
      ...
    </ThemeProvider>
  );
}
```

### **Use**

Import components you want into your UI

`import { Button } from ‘zahir-ui’;`

and use them as follow

```
  const example = () => (
    <div>
      <Button color="primary" onClick={() => 0}>Do something</Button>
      <Badge onClick={this.clear}>Cool</Badge>
    </div>
  )
```
